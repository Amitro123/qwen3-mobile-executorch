# Models

Place your trained model files here:

- `Qwen3-0.6B.Q4_K_M.gguf` - GGUF quantized model (~400MB)
- `tokenizer.json` - Qwen3 tokenizer (~11MB)

## How to get these files

1. Run the Kaggle notebook: `kaggle/qwen3-phone-deploy.ipynb`
2. After training completes, click "Save Version"
3. Download from the Output tab
4. Copy files here
